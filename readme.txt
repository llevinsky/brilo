# Brilo Webpage README

## Introduction
This README provides an overview of the Brilo webpage, a project developed using HTML, CSS, JavaScript, and Bootstrap. The development environment used for this project is Visual Studio Code.

## Features
- **Responsive Design:** The webpage is fully responsive and optimized for various screen sizes.
- **Interactive Elements:** Implemented using JavaScript to enhance user engagement.
- **Styling with Bootstrap & CSS:** Utilizes Bootstrap for pre-defined styles and custom CSS for unique aesthetics.

## File Structure
The project consists of the following main files:
- `index.html`: The main HTML file containing the structure of the webpage.
- `styles.css`: Contains all custom styles used to enhance Bootstrap's default styling.
- `script.js`: Includes JavaScript code that adds interactive features to the webpage.

## Getting Started
1. Clone the repository to your local machine.
2. Open the project folder in Visual Studio Code or your preferred code editor.
3. Preview `index.html` in a web browser to view the webpage.
